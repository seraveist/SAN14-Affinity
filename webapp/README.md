# 삼국지14 장수 편성기

정적 웹 앱입니다. `index.html`을 브라우저에서 열면 바로 실행됩니다.

## 고정된 로직

- 친애 관계는 `☆`, `◎`, `○`만 사용하고 `X`는 제외합니다.
- 버프 중첩은 같은 분파 안의 양방향/단방향 친애 연결 수로 계산합니다.
- 장수 점수는 `max(통솔+무력, 통솔+지력) + 개성점수`를 우선값으로 사용합니다.
- 개성 점수는 금색 `+5`, 파란색 `+2`, 빨간색 `+0`입니다.
- `악주`, `응원`은 추가 `+1`을 더합니다.
- 보유 장수 편성은 최대 10명 기준으로 친애 밀도와 장수 점수를 함께 보는 휴리스틱 분배입니다.
- 여무장은 기본적으로 기혼 상태로 편성합니다.
- 결혼 후보는 분파 점수에 반영하지 않고, `미혼 가정` 추천으로만 표시합니다.

## 데이터 갱신

```powershell

```
